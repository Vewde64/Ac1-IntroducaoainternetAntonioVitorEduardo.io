Iniciando sistema operacional

No caso do Windows, o Master Boot Record (MBR) verifica qual
partição do HD está ativa (configurada como Master) e inicializa
o “setor um” dela, essa área tem um código com a simples missão
de carregar o setor dois, essa parte varia de sistema para sistema.
Após isso o sistema faz a leitura de um arquivo de configuração
de boot, chamado de Boot Loader
