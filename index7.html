O Sistema operacional

A partir dele, é inicializado o núcleo (kernel), Assim como a BIOS estabelece
a ligação entre hardware e sistema, o kernel serve para firmar uma comunicação
estável entre hardware e software. O kernel carrega apenas os processos essencias
para inicialização do sistema, para assim não sobrecarregar o sistema, mas ao fim
disso a tela de escolha de usuarios é exibida e após o logon, os programas configurados
para iniciar junto com o sistema são carregados
